Sentimental Value
This project visualizes the first six months of Netflix's viewing data from 2025. Each layer moves from format to genre to language, scaled by hours watched or number of streams. Hover anywhere to explore.
Measures the total amount of time viewers spent watching a title. Calculated in total hours viewed across all viewers.
?

The sentiment matrix scores each title by analyzing its summary and keywords for emotional valence and narrative intensity, then weights those scores using basic engagement signals like view count and watch time. After normalization, those combined values become the two axes. Plotting them shows how the titles naturally cluster into the four viewing modes in the dataset.
